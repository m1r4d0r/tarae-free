타래 뽑기

💡 사용 방법

  • 마스토돈 / 트위터: 대화 타래 페이지에서 [우클릭] → [다른 이름으로 저장] (파일 형식: 웹페이지, 전부)으로 내려받은 HTML 파일을 업로드하세요. (트위터는 스크롤 캡쳐를 이어붙여주는 확장 프로그램의 정리된 HTML 출력도 함께 지원해요.)
  • 카카오톡: [메뉴] → [설정] → [대화 내용 내보내기] → [텍스트 메시지만 저장] (모바일) / [메뉴] → [대화 내용] → [대화 저장하기] (PC) 로 저장한 TXT 파일을 업로드하세요.
  • 디스코드: 대화 내용을 드래그로 복사(Ctrl+C)해서 아래 텍스트 창에 붙여넣기(Ctrl+V) 하세요.

⚠️ 주의 사항

  • 마스토돈·트위터는 사이트 구조가 바뀌면 추출이 실패할 수 있어요. 정상 작동하지 않으면 메인 페이지의 문의 방법으로 알려주세요.
  • 마스토돈·트위터는 HTML 저장 시 웹페이지, 전부가 아니라 다른 형식으로 저장하면 작동하지 않습니다.
  • 트위터는 플랫폼 특성상 긴 대화의 경우 한번에 전부 저장되지 않을 수 있습니다.
  • 카카오톡, 디스코드는 아이디를 임의 생성하고 프로필 이미지는 수집되지 않습니다. 뷰어에서 별도로 수정해주세요.
