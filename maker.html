👥 등장인물 프로필 설정

왼쪽 사이드바에서 타래(CSV) 파일을 불러와주세요.
[이름, 아이디, 타임스탬프(선택), 본문, 프로필이미지(선택)] 구조를 자동으로 인식합니다.

📖 사용 방법 가이드

• 실시간 편집: 대사 본문을 클릭하면 테두리가 나타나며 글자 편집이 가능합니다. 외부 빈 곳을 클릭하면 다시 깔끔하게 정돈됩니다.

• 인물 연동 교체: 미리보기 창의 [프로필 사진]이나 [이름] 영역을 클릭하면 지정 인물로 한 번에 변경 가능합니다.

• 행 합치기 / 분리: 각 줄 우측 도구에서 [⬆ 위 행과 합치기]로 바로 위 대사와 합치거나, 대사 칸에 커서를 두고 [✂ 커서에서 분리]를 누르면 그 위치 기준으로 두 줄로 나뉩니다.

• 카카오톡 레이아웃: 등장인물 프로필 설정에서 본인에 해당하는 인물의 이름을 정확히 "나"로 지정해야 오른쪽 말풍선(프로필 사진 없음)으로 표시됩니다. 다른 인물은 전부 왼쪽 말풍선으로 나옵니다. 대사에 줄바꿈이 있으면 말풍선이 여러 개로 이어붙어요.

• 아이디 표기: 이름 옆에 뜨는 아이디(@계정명)를 표시할지 숨길지 선택할 수 있어요. 트위터·마스토돈 레이아웃에 적용돼요 (디스코드·카카오톡은 원래부터 아이디를 표시하지 않아요).

• 부분 스크린샷: 구간 캡쳐 활성화 후 원하는 타깃 시작 행과 끝 행을 연달아 선택하면 깔끔한 크롭 캡쳐가 완료됩니다.
구간 지정 상태: 시작 대화 라인을 선택하세요.

캐릭터 프로필 상세 설정

⚠️ 원본 이미지 링크가 삭제되면 이미지가 보이지 않습니다. 이미지까지 보관하고 싶다면 별도 저장하고 업로드한 외부 링크(예: IMGUR)로 교체해주세요.
